# 📁 MediaExplorer – Android File Browser with Paging

**MediaExplorer** is a simple Android app that retrieves all media and document files from the device storage and displays them efficiently in a RecyclerView using **Paging**. Ideal for exploring internal files, testing file listing performance, or learning Android's storage access APIs.

---

## 🔍 Features

- 📂 Retrieve **all files** from the device storage
- 🧾 Display files in a **RecyclerView** with smooth scrolling
- ⚡ Uses **Paging** library for optimized performance
- 🔤 Show file details such as name, type, and path
- 📁 Support for media, documents, and other file types
- 💡 Clean and minimal UI

---

## 🛠️ Tech Stack

- **Kotlin**
- **RecyclerView**
- **Paging 3**
- **ViewModel & LiveData**
- **Storage Access (MediaStore & File APIs)**

---

## 🚀 Getting Started

### Prerequisites

- Android Studio
